Teddy – Keyboard operatoin

The following keybindings are predefined in teddy and can not be overridden:

  • Arrow keys (up, down, left, right) move the cursor around normally.
  • Shift+arrow keys selects text
  • Page Up and Page Down move the cursor halt a page up or half a page down.
  • End key moves the cursor to the end of the line.
  • Home key moves the cursor to the first non-whitespace character of the line. If you are already at the first non-whitespace character of the line moves to the first character of the line.
  • Delete key removes the selection, when it exists, or the character following the cursor.
  • Backspace key removes the selection, when it exists, or the character preceding the cursor.
  • Escape key enters command line mode.

No other keybindings are defined. You will have to use bindkey to bind them in the configuration file ($HOME/.config/teddy/rc). However the default configuration file created when no configuration file is found contains some useful keybindings, you are encouraged to read it.
