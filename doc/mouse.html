Teddy – Mouse operation

  • Editing area
    • Click to position cursor
    • Click and Drag to select text
    • Double-click and Drag to select words
    • Triple-click and Drag to select lines
    • Right click tries to interpret the text around the cursor as a file name and attempts to open. Also understands expressions like "<filename>:<linenumber>", opens <filename> and goes to <linenumber>. If the <filename> is a relative path it is resolved against the working directory of the buffer. If the specified file name can not be found it is used as the argument to search command (interactive search).
    • Right click on a selected region, same as right click if the selected region spans a single line, if it spans more than one line the region's text is evaluated as tcl.
  • Resize handle
    • Click and Drag
      • resizes the frame/column
    • Double click closes all the other frames in the column. If the frame is already the only one in the column, closes all the other columns in the window.
    • Right click opens a new frame in the column, or a new frame in another column if there is no space in the current column
    • Middle click closes the frame. If the frame is the last one in the column, closes the column (unless the column is the only existing column).
  • Tag
    • Click and Drag switches the buffers of the origin and target frames
    • Double click closes all the other frames in the column. If the frame is already the only one in the column, closes all the other columns in the window.
    • Right click opens a new frame in the column, or a new frame in another column if there is no space in the current column
    • Middle click closes the frame. If the frame is the last one in the column, closes the column (unless the column is the only existing column).
