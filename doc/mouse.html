Teddy – Mouse operation

  • Editing area
    • Click to position cursor
    • Click and Drag to select text
    • Double-click and Drag to select words
    • Triple-click and Drag to select lines
    • Right click: when a selection is active will display a menu of actions that can be executed on the selected text.
    • Scroll wheel scrolls the editing area.
  • Resize handle
    • Click and Drag resizes the frame/column
    • Click makes the frame bigger by reducing the space allocated to frames above and below it
  • Tag
    • Click and Drag moves the frame to the new position.
    • Double click minimizes all other frames in the column, same as clicking the up arrow button in the tag line.
    • Middle click closes the frame.

Context menu

  • Eval evaluates selection as if it was typed in the command line. Works with either scripts or external commands.
  • Copy text to command line
  • Open selected text treats selected text as a file name and tries to open it
  • Search text starts searching selected text in current file.
  • Search in TAGS if a TAGS file is loaded search selected text as a tag
